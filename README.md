# electron-vue

`Vue3 + Electron + webpack5 + typescript` 前端工程开发环境


## 安装依赖
```javascript
npm install
```

## `typescript`类型检测

```javascript
npm run tsc
```

## 运行 开发环境

```javascript
npm run dev
```

## 编译打包exe

```javascript
npm run build
```

## 目录说明
- `build` webpack config 配置目录
- `src` webpack编译输入目录，源代码
- `app` webpack编译输出目录 electron-builder 打包输入目录
- `dist` electron-builder 打包输出目录
- `typings` .d.ts目录
